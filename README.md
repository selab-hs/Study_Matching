## 🔨 개발 환경
- JAVA 17
- IDE: IntelliJ
- Framework: SpringBoot 3.0
- DB: MySQL, Redis
- ORM: JPA
- Thymeleaf
- Rest Docs

<br>

## 📌 기능 요구사항

1. 프로젝트(스터디) 생성

2. 프로젝트 팀원 모집
  
3. 프로젝트 관리 - 팀원 할일 인증 및 관리

4. 로그인 - oauth(깃허브, 구글 등…)

5. 인프라
